**1. two sum**
一开始想的是暴力

```python
class Solution:
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        first = 0
        while first < len(nums):
            for i in range(first+1, len(nums)):
                if nums[first] + nums[i] == target:
                  return [first, i]
            first += 1
 ```

```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        bufdic = {}
        for i in range(len(numbers)):
            if numbers[i] in bufdic:
                return [bufdic[numbers[i]]+1, i+1]
            else:
                bufdic[target-numbers[i]] = i
```

two-pointer的解法
```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        if not numbers:
            return []
        left, right = 0, len(numbers) - 1
        while left < right:
            if numbers[left] + numbers[right] == target:
                return [left+1, right+1]
            elif numbers[left] + numbers[right] > target:
                right -= 1
            else:
                left +=1
        return []
```                
